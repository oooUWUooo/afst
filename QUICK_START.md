# Quick Start Guide

Следуйте этим шагам, чтобы быстро запустить приложение и протестировать его функциональность.

## Шаг 1: Подготовка

Убедитесь, что у вас установлен Python 3.7+ и pip.

## Шаг 2: Установка зависимостей backend

```bash
pip install fastapi uvicorn python-multipart
```

## Шаг 3: Запуск backend сервера

```bash
uvicorn example_backend:app --reload
```

Сервер запустится на `http://localhost:8000`

## Шаг 4: Запуск frontend сервера

В новом терминале:

```bash
python start_server.py
```

Сервер запустится на `http://localhost:8080`

## Шаг 5: Настройка API URL

1. Откройте `http://localhost:8080` в браузере
2. Найдите раздел "API Configuration" (обычно в правом верхнем углу)
3. Введите URL: `http://localhost:8000`
4. Нажмите "Save URL"
5. Перезагрузите страницу (Ctrl+F5 или Cmd+R)

## Шаг 6: Регистрация и использование

1. Нажмите "Register" для создания нового аккаунта
2. Введите email и пароль
3. После успешной регистрации вы попадете в панель управления
4. Теперь вы можете:
   - Добавлять книги
   - Добавлять читателей
   - Выдавать и возвращать книги
   - Просматривать статистику

## Возможные проблемы и решения

### Если не удается подключиться к API:
- Убедитесь, что оба сервера запущены
- Проверьте, что вы ввели правильный URL API
- Попробуйте очистить кэш браузера

### Если страница не загружается:
- Проверьте, что порты 8080 и 8000 не заняты другими приложениями
- Убедитесь, что все зависимости установлены
